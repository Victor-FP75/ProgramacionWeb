# CEVICHE PERUANO

## Plato tipico en Perú

- Limón
- Pescado
- Ajos
- Sal
- Apio
- Cominos

1. Primero, lavar el pescado con sal.
2. Segundo, colocar el limón
3. Tercero, agregar trozos de pescado, agua minimo, sal, apio y licuar
4. Cuarto, agregar a la fuente del ceviche.
